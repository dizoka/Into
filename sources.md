Формула реалізована під логіку калькулятора Освіта.UA:
КБ = (К1×П1 + К2×П2 + К3×П3 + К4×П4) / (К1+К2+К3+(К4+K4max)/2) + ОУ,
після цього для 1–2 пріоритету застосовується галузевий коефіцієнт, якщо він є для конкретної пропозиції.
