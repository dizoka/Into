# ЛНТУ — калькулятор вступу 2026 на бюджет

Сайт для абітурієнтів ЛНТУ: вводиш лише середній бал НМТ, а сайт автоматично показує орієнтовний конкурсний бал і спеціальності/конкурсні пропозиції, на які можна пройти на бюджет.

## Що є

- один простий ввід: середній бал НМТ 100–200;
- автоматичний розрахунок орієнтовного конкурсного бала;
- для спеціальностей з особливою підтримкою автоматично показується коефіцієнт 1.02 для 1–2 пріоритету;
- список спеціальностей і пропозицій у стилі ЄДЕБО;
- тільки бюджетні орієнтири, без контракту;
- готово для GitHub + Render.

## Як запустити локально

Відкрий `index.html` у браузері.

## Render

Render Static Site:

- Build Command: пусто
- Publish Directory: `.`

Або використай `render.yaml`.

## Важливо

Фінальних прохідних балів 2026 ще немає, тому сайт використовує бюджетні пороги 2025 як орієнтир для вступу 2026.
