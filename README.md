# ЛНТУ 2026 — калькулятор як на Освіта.UA

Це не статична ручна таблиця. Сайт запускається як Node.js Web Service на Render і бере дані сервером з:

- `vstup.osvita.ua` — спеціальності, освітні програми, коефіцієнти предметів, статистика зарахування;
- `vstup2025.edbo.gov.ua` — статистика заяв за кодом конкурсної пропозиції, якщо сторінка доступна.

## Як запустити на Render

1. Завантаж файли в GitHub.
2. На Render натисни **New + → Web Service**.
3. Вибери репозиторій.
4. Settings:
   - Runtime: **Node**
   - Build Command: `npm install`
   - Start Command: `npm start`
5. Натисни Deploy.

Важливо: це **Web Service**, не Static Site, бо сайт підтягує актуальні сторінки Освіта.UA / ЄДЕБО через сервер.

## Формула

Калькулятор використовує формулу Освіта.UA / правил вступу 2026:

`КБ = ((К1×П1 + К2×П2 + К3×П3 + К4×П4 + КТ×ТК) / (К1 + К2 + К3 + (К4 + К4max)/2 + КТ) + ОУ) × РК × ГК`

На сайті ЛНТУ для Волинської області РК = 1.00.

ГК 1.02 застосовується тільки там, де в даних програми/категорії позначено особливу підтримку. Якщо дані не підтверджені, сайт не вигадує коефіцієнт.
