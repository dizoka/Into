# ЛНТУ — калькулятор конкурсного бала

Web Service для Render. Дані по пропозиціях, коефіцієнтах і статистиці бере з Вступ.ОСВІТА.UA, де сторінки позначені як отримані з ЄДЕБО.

## Render

- Type: Web Service
- Build Command: `npm install`
- Start Command: `npm start`

Після заміни файлів на GitHub зроби в Render: **Manual Deploy → Clear build cache & deploy**.

## Що виправлено

- Кожна конкурсна пропозиція зберігається окремо: спеціальність + назва пропозиції + форма навчання + ID.
- Мінімальний бюджетний бал береться з рядка **«Мінімальний рейтинговий бал серед зарахованих на бюджет»** на Освіта.UA, а не з іншої пропозиції.
- Для сторінок 2026, де ще немає фінальної статистики, сервер переходить на точну сторінку 2025 цієї ж пропозиції через посилання «2025» і бере статистику минулого року.
- Розрахунок конкурсного бала зроблено за формулою Освіта.UA з K4max.
