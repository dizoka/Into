# ЛНТУ · калькулятор конкурсного бала

Web Service для Render. Проєкт відкриває сайт на `/`, завантажує конкурсні пропозиції ЛНТУ з Освіта.UA, парсить окремі сторінки пропозицій і рахує конкурсний бал за формулою Освіта.UA.

## Render

- New → Web Service
- Build Command: `npm install`
- Start Command: `npm start`

Після оновлення файлів: Manual Deploy → Clear build cache & deploy.

## Що важливо

Кожна конкурсна пропозиція окрема: спеціальність + освітня програма + форма навчання + ID. Бюджетний поріг не змішується між різними пропозиціями однієї спеціальності.
