# ЛНТУ вступ 2026 — бюджетний орієнтир

Готовий статичний сайт для GitHub Pages або Render Static Site.

## Що всередині
- `index.html` — весь сайт в одному файлі.
- `render.yaml` — налаштування для Render.

## Оновлення даних
Список спеціальностей зроблено у стилі ЄДЕБО: код зліва, пошук, прокрутка, вибір спеціальності.
Фінальні прохідні бали 2026 ще не внесені, тому для бюджету використано реальні бюджетні пороги 2025 як орієнтир.

## Render
1. Завантаж файли в GitHub.
2. Render → New → Static Site.
3. Build Command: порожньо.
4. Publish Directory: `.`
5. Deploy.
