# ЛНТУ 2026 — розрахунок конкурсного бала

Оновлено: формула пріоритетів виправлена для всіх спеціальностей.

## Як рахує сайт

- 3–10 пріоритети: `(сума бал НМТ × коефіцієнт предмета) / (сума предметних коефіцієнтів + 0.05)`
- 1–2 пріоритети: результат 3–10 множиться на `1.02`, але тільки для спеціальностей, де є галузевий коефіцієнт.
- Якщо галузевого коефіцієнта немає, 1–2 і 3–10 будуть однакові.

Приклад J8 з балами 149 / 158 / 149 / іноземна 162:

- 3–10: 149.548
- 1–2: 152.539

Приклад H7 з балами 149 / 158 / 149 / іноземна 162:

- 3–10: 149.310
- 1–2: 152.297


## Перевірка формули 2026
Сайт рахує за формулою Освіта.UA/ЄДЕБО 2026: (K1×P1 + K2×P2 + K3×P3 + K4×P4) / (K1+K2+K3+(K4+K4max)/2) + ОУ. Для 1–2 пріоритетів застосовується ГК 1.02 лише там, де він є.
